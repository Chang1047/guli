安装vue
1.安装淘宝镜像cmd命令输入
npm install -g cnpm –registry=https://registry.npm.taobao.org

2.安装express
npm install -g express

3.安装vue
npm install vue

4.安装vue-cli
npm install  -g vue -cli

检验是否安装成功
vue -V
出现版本号即为成功！！！

5.从github上克隆vue配置文件
https://github.com/PanJiaChen/vue-element-admin

6.导入vscode
右键此文件选择在集成终端中打开

7.npm install 

8.npm run dev  后自动打开vue网页
